# Practical AI Workflows site

A premium, static marketing site for a small-business AI workflow implementation company.

## Positioning

- practical AI workflows for small businesses
- implementation first, not vague AI consulting
- start with one workflow, measure the result, then expand
- initial wedge: home services

## Pages

- Home
- Services
- How It Works
- Industries
- Trust and Safety
- About
- Contact

## Stack

- Astro
- TypeScript
- static GitHub Pages deployment via GitHub Actions

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run check
```
